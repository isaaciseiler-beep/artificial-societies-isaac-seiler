# Experiment Size Comparison

Each experiment uses the same 100 personas and the same 10 responses per persona-question. The only thing that changes is the number of aggregate trial runs.

The experiments use separate trial-id ranges, so they are independent runs.

| Size | Trial runs | Question-level runs | El-Sayed margin | Stevens margin | McMorrow margin |
| --- | ---: | ---: | ---: | ---: | ---: |
| 10% current | 100 | 300,000 | -6.5 | +17.4 | 0.0 |
| Current | 1,000 | 3,000,000 | -6.6 | +17.7 | -0.2 |
| 1000% current | 10,000 | 30,000,000 | -6.6 | +17.7 | -0.1 |
| 10000% current | 100,000 | 300,000,000 | -6.6 | +17.7 | -0.1 |

## Polling Check

## Read

The current 1,000-run size is already stable. Moving from 1,000 to 10,000 runs barely changes the margins, and moving from 10,000 to 100,000 does not change the one-decimal margins at all.

The model is closest to polling on McMorrow, somewhat close on El-Sayed compared with Detroit Chamber, and much more bullish on Stevens than either polling source.

Bottom line: the experiment size is not driving the finding. The main finding holds across all four sizes: Stevens looks strongest, El-Sayed weakest, and McMorrow close but uncertain.
