# Polling Check

This is the external check on the simulation.

Polling was not included in persona responses. The model answers first, then this file compares those answers with RCP and Detroit Chamber polling.

The comparison is useful because it shows where the persona model lines up with public polling and where it may be too soft or too hard on a candidate.

| Matchup | Weighted persona | RCP average | Detroit Chamber | Read |
| --- | ---: | ---: | ---: | --- |
| Abdul El-Sayed vs Mike Rogers | Abdul El-Sayed 31.3 / Rogers 37.9 / Undecided 30.8 | Abdul El-Sayed 43.3 / Rogers 43.0 / Unallocated 13.7 | Abdul El-Sayed 39.8 / Rogers 44.7 / Undecided 15.5 | weaker than polling for Democrat |
| Haley Stevens vs Mike Rogers | Haley Stevens 48.6 / Rogers 31.0 / Undecided 20.4 | Haley Stevens 43.7 / Rogers 43.3 / Unallocated 13.0 | Haley Stevens 41.5 / Rogers 43.8 / Undecided 14.7 | stronger than polling for Democrat |
| Mallory McMorrow vs Mike Rogers | Mallory McMorrow 32.0 / Rogers 32.1 / Undecided 35.8 | Mallory McMorrow 42.3 / Rogers 42.7 / Unallocated 15.0 | Mallory McMorrow 40.7 / Rogers 42.8 / Undecided 16.5 | broadly close to polling |

Bottom line: Stevens is the clearest general-election profile in this model. El-Sayed is the weakest. McMorrow is close to Rogers, but the high undecided share means I would not overread that matchup.
